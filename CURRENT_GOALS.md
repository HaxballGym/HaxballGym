# My goals for this project

- Make this library super clean. The architecture should be super modular and easy to understand. And we should have a super clean HOW_TO guide for new developers.
- Improve the UI, to make the replica of the game as close as possible to the original game (graphics, sound, extrapolation etc.) (can be done last)
- Find ways to improve the state of the art on Haxball, make the bot challenging against actual players.
- When training, I should have some kind of live view, where i can see the bot playing the game live. And we should train on multiple envs at once (like RLGym)
- Research the state of the art from RLGym, and implement it in this project.
- When everything is ready, update to 0.3.0 and publish it on PyPI, then archive other projects.
